# 🌾 Raitha-Varta – Flash Card Agriculture Advisor

Raitha-Varta is an Android application that transforms complex agricultural advisories into short, actionable flash cards for farmers. The app is designed for easy usability in field conditions and supports regional language accessibility.

## 🚀 Features

- 📱 Swipe-based flash card UI for quick learning  
- 🌾 Daily crop-specific farming tips  
- 🌐 Kannada + English language support  
- 🧠 AI-based advisory simulation  
- 🌱 Crop filtering (Paddy, Coconut, Tomato, etc.)  
- 📖 Real farmer success stories  
- 📶 Offline-ready experience  

---

## 🛠 Tech Stack

- **Language:** Kotlin  
- **UI:** Jetpack Compose  
- **Image Loading:** Coil  
- **Architecture:** Single Activity + Compose  
- **AI Integration:** Simulated GenAI-based advisory  

---

## 🎯 Impact

- Simplifies agricultural knowledge  
- Helps farmers take quick decisions  
- Supports precision farming  
- Improves accessibility of expert advice
