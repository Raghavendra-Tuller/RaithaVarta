# Raitha-Varta 🌾

Raitha-Varta is an Android app that converts agricultural advisories into short, swipeable flash cards for farmers.

## Features
- Daily crop tips
- Kannada + English support
- Swipe-based UI
- Crop filtering
- Success stories
- Expert query simulation

## Tech Stack
- Kotlin
- Jetpack Compose
- Coil
- GenAI
